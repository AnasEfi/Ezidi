Триггер
Триггером называется устройство, способное формировать два устойчивых значения выходного сигнала и скачкообразно изменять эти значения под действием внешнего управляющего сигнала.


В данной работе мы представим работу следующих триггеров:
1. асинхронный RS-триггер
(построим на логических элементах ИЛИ-НЕ по условию Задачи 1а,
на микросхеме JK по условию Задачи 1с)
2. синхронный RS-триггер
(построим на логических элементах И-НЕ по условию Задачи 1b)
3. D-триггер
(построим на логических элементах И-НЕ по условию Задачи 2a,
на микросхеме JK по условию Задачи 2b)
4. T-триггер
(построим на основе D-триггера на микросхеме JK по условию Задачи 3)
